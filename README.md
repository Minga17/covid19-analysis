# 🦠 COVID-19 Global Trends Analysis

> **Live Portfolio:** [portfolio.mingangolo.com](https://minga-portfolio.onrender.com)

A complete exploratory data analysis of global COVID-19 data covering 228 countries 
and 306,429 rows of daily case records.

---

## 📊 Key Findings

- 🌍 **78,843,232** total confirmed cases across 228 countries
- 💀 Global death rate of **2.12%** — 1,672,981 total deaths
- 🇫🇷 **France** was the most affected country with **5.9M** confirmed cases
- ⚠️ **Sudan** had the highest death rate at **7.4%**
- 💚 **83.2% recovery rate** — 65,604,778 people recovered globally
